Compressed files

Bisque ingest system supports most common compressed file formats and required operations.

  • Supported formats: Zip and Tar

    • Several operations can be applied after uncompressing the file:

  • Ingest all files as a dataset of separate images
  • Combine all files in a multi-dimensional image and ingest

    • Multi-dimensional images

    Files contained in the compressed package must comply to our specifications. Currently we require:

  • All files are located in the root
  • They should be sorted alphanumerically
  • 4D files should be in the Z-T order, where Z is most significative

    • Example of the 4D file:

  • sample_4d.zip
  • - T000_Z0.tif
  • - T000_Z1.tif
  • - T000_Z2.tif
  • - T001_Z0.tif
  • - T001_Z1.tif
  • - T001_Z2.tif
