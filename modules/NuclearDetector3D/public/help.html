Parameters

Nuclei size: We have approximately measured the size of the nucleus and obtained 3 microns. This value we will enter as "Nuclei Size" parameter.

Low intensity bound: We would also like to reject 20% of lower intensity levels as a valid response of a nuclei, so we set 0.2 as a "Low intensity level".

Pixel size: If the image has embedded meta-data then both pixel resolution X/Y and Z will be filled automatically. If you would want to change these values, they will be stored in the appropriate image tag and preserved.

Channels are selected by number and nuclear channel must to be present. You can also select a membrane channel if it exists, this could help to filter out incorrectly detected centroids.
